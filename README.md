# 🔐 Generador de Contraseñas 

App web de ciberseguridad que genera contraseñas seguras al instante.

### ✨ Features
- **Generación aleatoria:** Mayúsculas, minúsculas, números y símbolos
- **16 caracteres:** Longitud segura por defecto 
- **1 Click Copy:** Botón copiar con alert de confirmación
- **UI 10/10:** Diseño degradado rojo → negro, responsive

### 🛠️ Tech Stack
`HTML5` `CSS3` `JavaScript Vanilla`

### 💡 Cómo usar
1. Click en "Generar Contraseña" 
2. Click en "Copiar Contraseña"
3. Pégala donde la necesites
